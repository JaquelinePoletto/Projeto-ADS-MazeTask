Disposição da tela do Maze Task: 

1 - Opção 1: Usando Matrizes: Será utilizada uma matriz tridimensional, considerando 10 frases:

Frase[10][2][3]

Considerando uma frase com sujeito+verbo+objeto

Frase[0]['certo']['sujeito'] = "a menina"
Frase[0]['certo']['verbo'] = "comeu" 
Frase[0]['certo']['objeto'] = "a maçã"

Frase[0]['errado']['sujeito'] = "XXXX"
Frase[0]['errado']['verbo'] = "rugiu"
Frase[0]['errado']['objeto'] = "leite"

Frase[1]['certo']['sujeito'] = "eles"
Frase[1]['certo']['verbo'] = "pularam" 
Frase[1]['certo']['objeto'] = "corda"

Frase[1]['errado']['sujeito'] = "XXXX"
Frase[1]['errado']['verbo'] = "caminhou"
Frase[1]['errado']['objeto'] = "a voz"

Exibir um certo e um errado em cada posição randomicamente.

2 - Opção 2: Usando listas:

2.1 - Montar três listas com palavras ou expressões que possam formar uma frase no formato: sujeito+verbo+objeto
Ps: As frases acho que a Kelly já disponibilizou no github. Confirmar.

Deverão existir 3 listas para montar as frases corretas. Exemplo:
sujeito[]    verbo[]          objeto[]
A menina     comeu            maça
O sorvete    está             gelado
Nós          assistiremos     ao filme

Segundo a regra do Maze Task, a primeira linha sempre deverá conter a opção correta junto com um XXXXX

Constante  OpçãoIncorreta2[]  OpçaoIncorreta3[]
XXXXX      Rugiu              Leite
XXXXX      Voou               redonda 
XXXXX      fugiu              cinza

O prograna deverá montar a tela considerando:

Frase 1:
Linha 1: sujeito[1]   ou Constante XXXX       -> "A menina" ou "XXXXX"
Linha 2: verbo[1]     ou OpçãoIncorreta2[1]   -> "comeu" ou "Rugiu"
Linha 3: predicado[1] ou OpçãoIncorreta3[1]   -> "maça" ou "Leite"

Frase 2:
Linha 1: sujeito[2]   ou Constante XXXX       -> "O sorvete" ou "XXXXX"
Linha 2: verbo[2]     ou OpçãoIncorreta2[2]   -> "está" ou "Voou"
Linha 3: predicado[2] ou OpçãoIncorreta3[2]   -> "gelado" ou "redonda"

Frase 3:
Linha 1: sujeito[3]   ou Constante XXXX       -> "Nós" ou "XXXXX"
Linha 2: verbo[3]     ou OpçãoIncorreta2[3]   -> "assistiremos" ou "fugiu"
Linha 3: predicado[3] ou OpçãoIncorreta3[3]   -> "ao filme" ou "cinza"

2.2 - Implementar uma lógica para permitir mudar as opções corretas de lugar.

2.3 - A frase correta é obtida concatenando as listas sujeito[] + verbo[] + objeto[]

2.4 - A medida que o usuário selecionar uma opção, montar uma string concatenando os valores selecionados.
Para saber se o usuário marcou a opção correta, compare o conteúdo da string concatenada até o momento com o conteúdo da frase final esperada.
Se o valor estiver correto, pule para a próxima etapa. Se estiver incorreto, emitir uma mensagem de erro.

2.5 - Na etapa final, a frase montada pelo usuário deverá ser comparada com a string montada durante o processo.

2.6 - Se estiver incorreto, o usuário deverá selecionar novamente a opção correta;
    Se estiver correto, retomar o processo com a próxima frase da lista.
